# テキスト装飾記法 クイックリファレンス

JSONデータ内で使用できるテキスト装飾記法の一覧です。

## 📝 使用可能な記法

### 1. 改行
```
\n
```
例：`"問題1\\n問題2"` → 問題1 と 問題2 が別の行に表示されます

---

### 2. 下線
```
<u>テキスト</u>
```
例：`"下線部<u>Ａ</u>に関して"` → 下線部<u>Ａ</u>に関して

---

### 3. 太字
```
<strong>テキスト</strong>
または
<b>テキスト</b>
```
例：`"<strong>重要</strong>な項目"` → **重要**な項目

---

### 4. 斜体
```
<em>テキスト</em>
または
<i>テキスト</i>
```
例：`"<em>会計基準</em>に基づく"` → *会計基準*に基づく

---

### 5. 下線 + 太字の組み合わせ
```
<u><strong>テキスト</strong></u>
```
例：`"下線部<u><strong>Ａ</strong></u>に関して"` → 下線部<u><strong>Ａ</strong></u>に関して

---

### 6. 色付きテキスト
```
<span style="color: red;">テキスト</span>
<span style="color: blue;">テキスト</span>
```
例：`"空欄<span style=\"color: red;\">①</span>に答えなさい"`

使用可能な色：
- `red` (赤)
- `blue` (青)
- `green` (緑)
- `orange` (オレンジ)
- など

---

### 7. 背景色付きテキスト（ハイライト）
```
<span style="background-color: yellow;">テキスト</span>
```
例：`"重要：<span style=\"background-color: yellow;\">必須項目</span>"`

---

### 8. 組み合わせ例

```json
{
  "question": "下線部<u><strong>Ａ</strong></u>の<em>給付算定式基準</em>に関して、<span style=\"color: red;\">期間定額基準</span>との違いを説明しなさい。"
}
```

---

## 🎯 実用例

### 問題文での使用例

```json
{
  "id": "No.2",
  "question": "問題1-問1 問1 我が国の退職給付会計の開示に関する以下の設問に答えなさい。\\n退職給付債務の算定にあたり、退職給付見込額を当連結会計年度末までの期間に\\n帰属させる方法については、<u><strong>Ａ</strong></u> 給付算定式基準によっております。\\n1．下線部<u><strong>Ａ</strong></u>に関して、期間定額基準との違いに触れながら、<strong>給付算定式基準</strong>とは何かを説明しなさい。"
}
```

---

## ⚠️ 注意事項

1. **JSON内でのエスケープ**
   - JSON内では`"`（ダブルクォート）は`\"`でエスケープする必要があります
   - 例：`"<span style=\"color: red;\">テキスト</span>"`

2. **改行と組み合わせ**
   - 改行（`\n`）とHTMLタグは組み合わせて使用できます
   - 例：`"問題1\\n<u>下線部</u>\\n問題2"`

3. **タグの入れ子**
   - HTMLタグは正しく閉じる必要があります
   - 例：`<u><strong>テキスト</strong></u>` ✅
   - 例：`<u><strong>テキスト</u></strong>` ❌（間違い）

---

## 📋 使用可能なHTMLタグ一覧

現在実装されているタグ：

- `<u>` - 下線
- `<strong>`, `<b>` - 太字
- `<em>`, `<i>` - 斜体
- `<span style="...">` - インラインスタイル（色、背景色など）

他のタグを追加する場合は、実装を更新する必要があります。
